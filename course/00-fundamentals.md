# Модуль 0. Фундамент 🟢

**Срок:** 2–3 недели  
**Уровень:** Junior  
**Цель модуля:** заложить базу, без которой Go превратится в магию. Терминал, Git, HTTP, базовые структуры данных и алгоритмы.

---

## Урок 0.1. Терминал как родной

**🎯 Цель:** свободно двигаться по файловой системе без мышки.

**📚 Теория:**
- Навигация: `cd`, `ls -la`, `pwd`, `tree`
- - Просмотр: `cat`, `less`, `head`, `tail -f`
  - - Пайпы и перенаправление: `|`, `>`, `>>`, `2>&1`, `/dev/null`
    - - Поиск: `grep -rn`, `find . -name`, `xargs`
      - - Права: `chmod`, `chown`, `umask`
        - - Окружение: `export`, `$PATH`, `~/.zshrc` / `~/.bashrc`
         
          - **💻 Практика:** напиши `cleanup.sh`, который ищет `.log`-файлы старше 7 дней в указанной директории и удаляет их с подтверждением.
         
          - **✅ Чек:** найди все строки `TODO` в проекте и сохрани в файл одной командой.
         
          - **⚠️ Красный флаг:** открываешь Finder/Explorer, чтобы перейти в папку.
         
          - ---

          ## Урок 0.2. Git и GitHub реально

          **🎯 Цель:** понять модель Git, а не три магические команды.

          **📚 Теория:**
          - Объектная модель: blob, tree, commit, tag
          - - HEAD, индекс, рабочая копия
            - - Команды: `init`, `add`, `commit`, `push`, `pull`, `fetch`, `branch`, `switch`, `merge`, `rebase`, `rebase -i`, `cherry-pick`, `stash`, `reflog`, `reset --hard/--soft`
              - - Разрешение конфликтов
                - - Pull Request, fork-flow vs trunk-based
                 
                  - **💻 Практика:** создай репо `go-journey`, ветку `week-01`, 5 коммитов, squash в один через `rebase -i`, открой PR в `main`, намеренно вызови конфликт и разреши его.
                 
                  - **✅ Чек:** объясни разницу между `merge` и `rebase` и когда применять каждое.
                 
                  - **⚠️ Красный флаг:** делаешь `git push --force` на `main`.
                 
                  - ---

                  ## Урок 0.3. HTTP, REST, JSON

                  **🎯 Цель:** понимать, что летит по проводам.

                  **📚 Теория:**
                  - Методы: GET, POST, PUT, PATCH, DELETE — идемпотентность и безопасность
                  - - Коды: 1xx, 2xx, 3xx, 4xx, 5xx
                    - - Заголовки: `Content-Type`, `Authorization`, `Cache-Control`, `ETag`, `Cookie`
                      - - CORS, preflight
                        - - REST: уровни Ричардсона, HATEOAS
                          - - JSON: типы, валидация
                           
                            - **💻 Практика:** через `curl -v` сходи в GitHub API — получи свой профиль, создай gist, удали gist. Прочитай каждый заголовок ответа.
                           
                            - **✅ Чек:** объясни, чем PUT отличается от PATCH и почему GET не должен менять состояние.
                           
                            - ---

                            ## Урок 0.4. Структуры данных на бумаге

                            **🎯 Цель:** уметь нарисовать и объяснить.

                            **📚 Теория:** массив, динамический массив, односвязный/двусвязный список, стек, очередь, дек, хэш-таблица (открытая адресация vs цепочки), BST, куча, граф (списки смежности vs матрица).

                            **💻 Практика:** реализуй на любом языке стек и очередь — обе версии: на массиве и на связном списке.

                            **✅ Чек:** объясни, почему `map` обычно O(1), но в худшем случае O(n).

                            ---

                            ## Урок 0.5. Алгоритмы-минимум

                            **📚 Теория:** бинарный поиск, два указателя, sliding window, сортировки (bubble, merge, quick), рекурсия и хвостовая рекурсия, нотация O(n).

                            **💻 Практика:** 20–30 задач LeetCode Easy на любом языке. Главное — решать самому, подсматривать только после часа борьбы.

                            **✅ Чек:** реализуй бинарный поиск с нуля без подглядывания, объясни инварианты цикла.

                            ---

                            ## Урок 0.6. Сети на пальцах

                            **📚 Теория:** OSI/TCP-IP, TCP vs UDP, three-way handshake, DNS-резолвинг, TLS-рукопожатие, HTTPS, HTTP/2 vs HTTP/3.

                            **✅ Чек:** объясни, что происходит при вводе `https://google.com` в браузер — от DNS до рендеринга.

                            ---

                            ## Урок 0.7. Рабочая среда

                            **📚 Теория:** установка Go (через `gvm`/`asdf` или официальный инсталлер), VS Code или GoLand, расширение Go, `golangci-lint`, `.editorconfig`, `pre-commit` хуки.

                            **💻 Практика:** настрой workspace, установи линтер, проверь `go version`, `go env`.
